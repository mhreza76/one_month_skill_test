### basic comands of Angular CLI

#### create new project
```cli
ng new project-name
```

#### run project
```cli
ng serve
```

#### add bootstrap on a project
```cli
ng add @ng-bootstrap/ng-bootstrap
```

#### add Material UI
```CLI
    ng add @angular/material
```

#### add Material UI
```CLI
    ng g directive directive-name
```

#### add Material UI
```CLI
    ng g service service-name
```
